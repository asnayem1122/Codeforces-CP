# 📘 Codeforces-CP

### 🔎 Overview
This repository contains my solutions to various **Codeforces problem sets**, written in C++.  
It serves as a practice archive for competitive programming and algorithmic problem solving.

### 📂 Repository Structure
- Each file is a solution to a Codeforces problem.  
- **Note:** File names here are **not based on problem codes** (like `1472A.cpp`).
- Instead, they follow the problem name (e.g., `cards_for_friends.cpp`, `watermelon.cpp`).  
- To find the corresponding problem, check the comments at the top of each file — I include the Codeforces problem link or ID there.

### 🚀 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/asnayem1122/Codeforces-CP.git
   ```
2. Navigate to the file you want to test.
3. Compile and run:
   ```bash
   g++ filename.cpp -o solution
   ./solution
   ```

### 🧩 Example
For **Cards for Friends (Problem 1472A)**:
```cpp
// Input
5
2 2 3
3 3 2
5 10 2
11 13 1

// Output
YES
NO
YES
```
